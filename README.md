# Retail Transaction Dashboard

## Overview
Retail Transaction Dashboard is a data analysis and visualization project designed to monitor retail sales performance using transaction-level data.  
The dashboard helps stakeholders understand sales trends, product performance, and key revenue metrics through interactive visualizations.

This project focuses on transforming raw transactional data into actionable business insights and presenting them in a clear, stakeholder-friendly dashboard using Looker Studio.

---

## Objectives
- Monitor overall sales performance over time
- Identify top-performing and underperforming products
- Track key sales metrics to support data-driven decision making
- Provide an intuitive dashboard for business stakeholders

---

## Dataset Description
The dataset consists of retail transaction data, including:
- Transaction date
- Product information
- Sales amount
- Quantity sold

> **Note:**  
> Raw data and processing notebooks are no longer available, as this project was completed earlier.  
> The focus of this repository is on the analytical approach, dashboard design, and business insights derived from the data.

---

## Key Metrics & KPIs
The dashboard highlights several core retail KPIs, including:
- Total Revenue
- Number of Transactions
- Average Order Value (AOV)
- Product-level Sales Performance
- Sales Trends Over Time

---

## Analysis & Insights
The analysis focuses on:
- Sales trend analysis to detect growth or decline patterns
- Product performance comparison to identify revenue contributors
- Time-based analysis to observe changes in sales volume and revenue

Insights from the dashboard are intended to help stakeholders:
- Prioritize high-performing products
- Detect periods of sales fluctuation
- Support planning and performance monitoring activities

---

## Dashboard
The final output of this project is an interactive dashboard built with Looker Studio.  
The dashboard allows users to:
- Explore sales performance over time
- Compare product-level metrics
- Quickly identify key trends and patterns
